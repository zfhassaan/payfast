## Documentation Structure

All documentation files are located in the `content/` subdirectory:

- Markdown files (`.md`) - Documentation content
- `index.html` - Interactive documentation viewer
- `app.js` - Documentation viewer application
- `style.css` - Documentation styling
- `mascot.png` - Package logo/mascot

## Viewing Documentation

### Option 1: Interactive Viewer

Open `index.html` in a web browser to view the interactive documentation with search, navigation, and syntax highlighting.

### Option 2: Direct Markdown Files

Browse the `content/` directory and open any `.md` file directly in your markdown viewer or editor.
